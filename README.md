# reactjs-business-onepage

## TechStack :
- React JS
- Vite
- Tailwind CSS
- DaisyUI

## Getting Started :
- Go to the client folder
- Type "npm i" in the terminal to install dependencies
- Type "npm run dev" in the terminal
- Open localhost in browser : [http://localhost:5173/](http://localhost:5173/)
